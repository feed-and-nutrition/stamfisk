Raw data (Vitamin D, E & K)

Raw data of vitamin D, E, and K samples

About this page

This page contains a raw data table with the following options:

  • Copy: Copy the selected data to clipboard.

  • Download: Download the whole data either in “CSV” or “Excel” format.

  • Show x entries: Change the number of entries (x) shown in the table where x can be one of 10, 50, 100 or all rows (All).

  • Search: Enter a search term to filter the table.

  • Filter boxes: Choose an item from dropdown box to filter the table.

Values

The Stamfisk site provides two sets of values, both of which represent measured levels of nutrients/metabolites.

  1. Orig: Original value in character type. It contains the original <LOQ values.
  2. Value: Converted value in numeric type (double). All <LOQ values are converted to numeric values using a random imputation approach (see Imputation of LOQ values for details)
